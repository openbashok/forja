You are a security tool developer. Generate a JavaScript Object Sniffer — a runtime instrumentation script that intercepts, monitors, and logs all interesting JavaScript objects, variables, API calls, and data flows in a web application.

This is Forja's most important analysis tool. It gives the penetration tester X-ray vision into the application's runtime behavior.

## What the Sniffer MUST Monitor

### 1. Global Variables & Objects
- Enumerate all non-default properties on `window` (compare against a baseline of standard browser globals)
- For each custom global: log its name, type, and a preview of its value
- Use `Object.defineProperty()` to watch for changes on interesting globals (those containing tokens, keys, user data, config)
- Watch for new globals being added using a periodic scan or Proxy on window

### 2. Storage Operations
- Monkeypatch `Storage.prototype.setItem` and `Storage.prototype.getItem` and `Storage.prototype.removeItem`
- Log every localStorage/sessionStorage operation: key, value, operation type
- Flag sensitive-looking keys: token, session, auth, key, secret, password, user, jwt, csrf, api

### 3. Cookie Operations
- Hook `document.cookie` setter using Object.defineProperty on Document.prototype
- Log every cookie write: name, value, flags
- Flag cookies without Secure/HttpOnly/SameSite attributes (when visible from JS)

### 4. Network Calls (fetch + XHR)
- Monkeypatch `window.fetch` — log method, URL, headers (especially Authorization, X-API-Key, Cookie), request body, response status, content-type
- Monkeypatch `XMLHttpRequest.prototype.open` and `.send` — same details
- If jQuery is present, also hook `$.ajax`
- If axios is present, add request/response interceptors
- Flag requests containing tokens, credentials, or sensitive data in URL params or body
- Show response body preview for JSON responses (first 500 chars)

### 5. Form Submissions
- Hook all form submit events with addEventListener on document
- Log: form action, method, all field names and values (especially password fields, hidden inputs)
- Flag forms without CSRF tokens

### 6. postMessage / Cross-Origin Communication
- Hook `window.addEventListener('message', ...)` and `window.postMessage`
- Log: origin, data payload, target origin
- Flag messages accepted from wildcard origins ('*')

### 7. DOM Mutations on Sensitive Elements
- Use MutationObserver on document.body to watch for:
  - New hidden input fields (type="hidden") with token-like names
  - Password fields being populated programmatically
  - Elements with data attributes containing tokens or keys
  - Script tags being dynamically added

### 8. Eval and Dynamic Code
- Hook `eval()`, `new Function()`, `setTimeout(string)`, `setInterval(string)`
- Log the code being dynamically evaluated (truncated to 200 chars)
- Flag as potential security risk

```javascript
// Forja Object Sniffer panel
(function() {
  if (document.getElementById('forja-panel')) {
    document.getElementById('forja-panel').style.display = 'block';
    return;
  }

  const panel = document.createElement('div');
  panel.id = 'forja-panel';
  panel.style.cssText = 'position:fixed;bottom:20px;right:20px;width:600px;height:500px;' +
    'background:#2b2b2b;color:#d4d4d4;border:1px solid #555;border-radius:4px;z-index:999999;' +

  const controls = document.createElement('span');
  controls.style.cssText = 'display:flex;gap:8px;align-items:center;';

  // Event counter badge
  const badge = document.createElement('span');
  badge.id = 'forja-badge';
  badge.style.cssText = 'background:#e8a238;color:#1e1e1e;padding:1px 8px;border-radius:10px;font-size:10px;font-weight:bold;';
  badge.textContent = '0';

  const minBtn = document.createElement('span');
  minBtn.textContent = '—';
  minBtn.style.cssText = 'cursor:pointer;color:#888;font-size:14px;';

  const closeBtn = document.createElement('span');
  closeBtn.textContent = '✕';
  closeBtn.style.cssText = 'cursor:pointer;color:#888;font-size:14px;';
  closeBtn.onclick = () => panel.style.display = 'none';

  let minimized = false;
  minBtn.onclick = () => {
    minimized = !minimized;
    tabBar.style.display = minimized ? 'none' : 'flex';
    logArea.style.display = minimized ? 'none' : 'block';
    toolbar.style.display = minimized ? 'none' : 'flex';
    statusBar.style.display = minimized ? 'none' : 'flex';
    panel.style.height = minimized ? 'auto' : '500px';
    panel.style.resize = minimized ? 'none' : 'both';
  };

  controls.appendChild(badge);
  controls.appendChild(minBtn);
  controls.appendChild(closeBtn);
  titleBar.appendChild(controls);
  panel.appendChild(titleBar);

  // Drag
  let isDragging = false, dragX = 0, dragY = 0;
  titleBar.addEventListener('mousedown', (e) => {
    if (e.target === closeBtn || e.target === minBtn) return;
    isDragging = true;
    dragX = e.clientX - panel.offsetLeft;
    dragY = e.clientY - panel.offsetTop;
    document.addEventListener('mousemove', onDrag);
    document.addEventListener('mouseup', () => { isDragging = false; document.removeEventListener('mousemove', onDrag); }, { once: true });
  });
  function onDrag(e) {
    if (!isDragging) return;
    panel.style.left = (e.clientX - dragX) + 'px';
    panel.style.top = (e.clientY - dragY) + 'px';
    panel.style.right = 'auto';
    panel.style.bottom = 'auto';
  }

  // Tab bar for filtering by category
  const tabBar = document.createElement('div');
  tabBar.style.cssText = 'display:flex;background:#333;border-bottom:1px solid #444;flex-shrink:0;overflow-x:auto;';
  const tabs = ['All', 'Globals', 'Network', 'Storage', 'Cookies', 'Forms', 'PostMsg', 'DOM', 'Eval'];
  let activeTab = 'All';
  tabs.forEach(t => {
    const tab = document.createElement('span');
    tab.textContent = t;
    tab.dataset.tab = t;
    tab.style.cssText = 'padding:4px 10px;cursor:pointer;font-size:11px;color:#888;border-bottom:2px solid transparent;';
    if (t === 'All') { tab.style.color = '#e8a238'; tab.style.borderBottomColor = '#e8a238'; }
    tab.onclick = () => {
      activeTab = t;
      tabBar.querySelectorAll('span').forEach(s => { s.style.color = '#888'; s.style.borderBottomColor = 'transparent'; });
      tab.style.color = '#e8a238';
      tab.style.borderBottomColor = '#e8a238';
      filterLog();
    };
    tabBar.appendChild(tab);
  });
  panel.appendChild(tabBar);

  // Toolbar
  const toolbar = document.createElement('div');
  toolbar.style.cssText = 'padding:3px 8px;background:#333;border-bottom:1px solid #444;display:flex;gap:6px;flex-shrink:0;align-items:center;';

  const pauseBtn = document.createElement('button');
  pauseBtn.textContent = 'Pause';
  pauseBtn.style.cssText = 'background:#555;color:#d4d4d4;border:1px solid #666;padding:2px 10px;border-radius:3px;cursor:pointer;font-size:11px;';
  let paused = false;
  pauseBtn.onclick = () => { paused = !paused; pauseBtn.textContent = paused ? 'Resume' : 'Pause'; pauseBtn.style.background = paused ? '#e8a238' : '#555'; pauseBtn.style.color = paused ? '#1e1e1e' : '#d4d4d4'; };

  const clearBtn = document.createElement('button');
  clearBtn.textContent = 'Clear';
  clearBtn.style.cssText = 'background:#555;color:#d4d4d4;border:1px solid #666;padding:2px 10px;border-radius:3px;cursor:pointer;font-size:11px;';

  exportBtn.textContent = 'Export JSON';
  exportBtn.style.cssText = 'background:#555;color:#d4d4d4;border:1px solid #666;padding:2px 10px;border-radius:3px;cursor:pointer;font-size:11px;';
  exportBtn.onclick = exportLog;

  const searchInput = document.createElement('input');
  searchInput.placeholder = 'Filter...';
  searchInput.style.cssText = 'background:#1e1e1e;color:#d4d4d4;border:1px solid #555;padding:2px 6px;border-radius:3px;font-size:11px;flex:1;min-width:80px;';
  searchInput.oninput = filterLog;

  toolbar.appendChild(pauseBtn);
  toolbar.appendChild(clearBtn);
  toolbar.appendChild(exportBtn);
  toolbar.appendChild(searchInput);
  panel.appendChild(toolbar);

  // Log area
  const logArea = document.createElement('div');
  logArea.id = 'forja-log';
  logArea.style.cssText = 'flex:1;padding:4px;overflow-y:auto;overflow-x:hidden;background:#1e1e1e;';
  panel.appendChild(logArea);

  // Status bar
  const statusBar = document.createElement('div');
  statusBar.id = 'forja-status';
  statusBar.style.cssText = 'padding:3px 10px;background:#3c3f41;color:#888;font-size:10px;border-top:1px solid #555;' +

  statusBar.innerHTML = '<span>Monitoring active</span><span>Forja Agent</span>';
  panel.appendChild(statusBar);

  document.body.appendChild(panel);

  // --- Internal state ---
  let eventCount = 0;
  const allEntries = [];

  function filterLog() {
    const search = searchInput.value.toLowerCase();
    logArea.querySelectorAll('.forja-entry').forEach(el => {
      const matchTab = activeTab === 'All' || el.dataset.category === activeTab;
      const matchSearch = !search || el.textContent.toLowerCase().includes(search);
      el.style.display = (matchTab && matchSearch) ? 'block' : 'none';
    });
  }

  // --- Logging helpers ---
  window.forjaSniff = function(category, msg, detail, type) {
    if (paused) return;
    eventCount++;
    badge.textContent = eventCount;

    const colors = {
      info: '#8acdff', success: '#6a9955', warn: '#e8a238',
      error: '#f44747', critical: '#ff6b6b', data: '#d4d4d4', header: '#c586c0'
    };
    const color = colors[type] || colors.data;
    const time = new Date().toLocaleTimeString();

    const entry = document.createElement('div');
    entry.className = 'forja-entry';
    entry.dataset.category = category;
    entry.style.cssText = 'margin:1px 0;padding:2px 4px;border-bottom:1px solid #2a2a2a;word-break:break-all;';

    const catBadge = '<span style="background:#333;color:#888;padding:0 4px;border-radius:2px;font-size:10px;margin-right:4px;">' + category + '</span>';
    entry.innerHTML = '<span style="color:#555;">[' + time + ']</span> ' + catBadge + '<span style="color:' + color + ';">' + escapeHtml(msg) + '</span>';

    if (detail) {
      const detailEl = document.createElement('div');
      detailEl.style.cssText = 'margin-left:20px;color:#888;font-size:10px;max-height:60px;overflow:hidden;cursor:pointer;';
      detailEl.textContent = typeof detail === 'object' ? JSON.stringify(detail, null, 2).substring(0, 500) : String(detail).substring(0, 500);
      detailEl.onclick = () => { detailEl.style.maxHeight = detailEl.style.maxHeight === 'none' ? '60px' : 'none'; };
      entry.appendChild(detailEl);
    }

    logArea.appendChild(entry);
    logArea.scrollTop = logArea.scrollHeight;

    allEntries.push({ time, category, msg, detail, type });
    console.log('[Forja:' + category + '] ' + msg, detail || '');
  };

  function escapeHtml(s) {
    return String(s).replace(/&/g, '&amp;').replace(/</g, '&lt;').replace(/>/g, '&gt;');
  }

  function exportLog() {
    const blob = new Blob([JSON.stringify(allEntries, null, 2)], { type: 'application/json' });
    const a = document.createElement('a');
    a.href = URL.createObjectURL(blob);
    a.download = 'forja_sniffer_' + Date.now() + '.json';
    a.click();
  }
})();
```

## Important Rules

1. Use the panel template above as-is — do NOT simplify or remove features (tabs, toolbar, drag, resize)
2. After the panel setup, implement ALL the monitoring hooks described above
3. Use `forjaSniff(category, message, detail, type)` for ALL logging. Categories must match the tab names: 'Globals', 'Network', 'Storage', 'Cookies', 'Forms', 'PostMsg', 'DOM', 'Eval'
4. The sniffer must be NON-DESTRUCTIVE — it must not break the application's functionality. All hooks must call the original function after logging.
5. Wrap each hook in try-catch so one failure doesn't prevent others from running
6. Save original functions before overriding (e.g., `const origFetch = window.fetch`)
7. For globals enumeration, exclude standard browser properties — compare against a baseline list
8. Flag sensitive data patterns: passwords, tokens (JWT, Bearer), API keys, secrets, SSNs, emails, credit card numbers
9. The script should work on any modern browser (Chrome, Firefox, Edge)
10. Keep the total script under 300 lines — focus on quality of monitoring over quantity of code
11. Update the status bar with the count of intercepted events
12. Use appropriate log types: 'warn' for sensitive data exposure, 'error' for security issues, 'info' for normal operations

## Application Context Usage

When given application context (endpoints, auth patterns, tech stack):
- Focus monitoring on the specific auth mechanisms used (Bearer tokens, cookies, API keys)
- Watch for the specific global variables and storage keys the application uses
- Tailor form monitoring to the application's authentication forms
- If jQuery/axios/specific frameworks are detected in the tech stack, hook their specific APIs
